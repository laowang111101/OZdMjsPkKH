# 基于Python的Django-vue宾馆管理信息系统 python990

## 项目概述

本项目是基于Python的Django框架与vue前端框架开发的宾馆管理信息系统。系统包含管理员与用户前台两个部分，具备完善的功能模块，能够满足宾馆管理的日常需求。

## 技术栈

- 编程语言: Python
- 开发工具: Pycharm
- 后端框架: Django
- 前端框架: vue
- 数据库版本: MySQL5.7+
- 数据库工具：Navicat、SQLyog
- 关键技术：HTML、B/S架构

## 功能模块

### 管理员功能

- 个人中心
- 会员管理
- 客房类型管理
- 热门客房管理
- 客房预约管理
- 系统管理

### 用户前台功能

- 热门客房浏览
- 优惠资讯查看
- 个人中心
- 后台管理（特定权限）

## 系统角色

- 管理员：负责系统的维护与宾馆业务管理
- 用户：浏览宾馆信息，进行房间预约等

## 运行环境

- Python环境
- Django框架支持
- vue前端框架
- MySQL数据库
- B/S架构

## 部署步骤

（部署步骤根据实际情况编写，以下仅为示例）

1. 环境配置：确保Python环境、MySQL数据库等已正确安装
2. 代码部署：将后端代码部署至服务器
3. 数据库导入：使用Navicat或SQLyog导入数据库结构及数据
4. 前端部署：将vue构建的前端代码部署至Web服务器
5. 系统配置：配置服务器，确保前后端可以正常交互
